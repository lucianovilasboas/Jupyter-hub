# Curso Completo de Estatística

**Baseado no Tutorial de Estatística da W3Schools — do básico ao avançado, com exemplos práticos em Python**

---

## Apresentação

Bem-vindo ao **Curso Completo de Estatística**! Este material foi cuidadosamente elaborado para servir como um livro interativo de Estatística, adaptado para estudantes brasileiros, com exemplos práticos em Python.

São 42 capítulos em português, organizados em 4 módulos, cobrindo desde conceitos fundamentais até testes de hipótese e tabelas de consulta.

## Objetivos

- Compreender os conceitos fundamentais da Estatística
- Dominar técnicas de estatística descritiva e inferencial
- Aprender a visualizar dados de forma eficaz
- Realizar testes de hipótese e estimações
- Aplicar estatística com Python (NumPy, SciPy, Matplotlib)

## Pré-requisitos

- Computador com Python 3.10 ou superior
- Conhecimento básico de Python (variáveis, listas, funções)
- Bibliotecas: numpy, scipy, matplotlib, pandas

> Se você não conhece Python, recommendedo fazer o Curso de Python deste hub antes.

## Como Utilizar

1. Siga os capítulos na ordem numérica (S00 a S42)
2. Leia atentamente cada explicação antes de executar os exemplos
3. Execute as células no seu próprio ambiente Jupyter
4. Faça os exercícios propostos sem consultar a resposta primeiro
5. Desafie-se com os exercícios marcados como "Difícil"
